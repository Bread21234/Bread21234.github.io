Considered ISPs

Eagle Card

Link to Proposal

The Eagle Card is a project which will create a card which would allow the user to swipe it near a device to open a predetermined web-page in the browser. I was first enticed by this isp because I find it very unique and I believe a card like the Eagle Card could have uses. However, I rejected this project because I overall believe that it would be more cumbersome to use a physical object than to simply type a link. The card could be easily replaced with a simple bookmark.

AI Chatbot

Link to Proposal

This AI chatbot is designed to help children and teens process their emotions through meaningful dialogue with this chatbot. This project sparked my interest because I believe mental health is a very important part of a person’s overall health, and creating a chatbot for that would be amazing. Additionally, I would definitely be interested in using AI to create a bot that could respond to a user and meet their emotional needs. What caused me to decline this project is that I do not feel AI has progressed far enough yet to put the mental health of children in its hands. Mental health is a very volatile aspect of health and if the AI made a mistake, then there could be legitimate negative consequences. Therefore, I decided to decline the project.

Bus Tracker

Link to Proposal

This proposal is for an isp which would track school buses. With this information, it could estimate times of bus arrival, and in the event the bus is running late, provide new times. I considered this isp because I feel it would be extremely beneficial between the high school and the STEAM center. However, the issue with this project is that bus drivers could simply use the radio to communicate their location. Therefore, I feel the isp could be a little redundant and unnecessary.

FreeCash

Link to Proposal

FreeCash is a program designed to predict stock prices and trade in order to gain a profit. It would use data from Twitter, Google, and r/wallstreetbets. I was interested in this project due to its use of AI. I am very interested in AI and machine learning and I feel this would be a great opportunity to put those concepts into practice. However, the main drawback for me is the sources where the AI would receive its information. Twitter is not exactly known for its trustworthiness and neither is Reddit, so I have declined this project because I believe the project could source its information better.

Screen Privacy

Link to Proposal

This ISP is designed to fight the issue of privacy on screens in real life by detecting when others look at your screen and turning it off in response. I really like this project because I feel it’s fun, unique, and I would enjoy working with facial recognition. For me, the main drawback is the response to other people looking at your screen; I believe it would be more inconvenient for me if my screen started turning off while I was using it.

Dx3

Link to Proposal

Dx3 is a project designed to help combat drowsiness and keep the user awake. It works by using facial tracking to determine whether or not the user is awake. If Dx3 determines that the user has fallen asleep, it supplies them with a stimulus such as music, a buzzer, or a small electrical shock. I really enjoy the idea of this project and I feel it is very unique. Additionally, I feel the facial recognition aspects would challange me and I would greatly enjoy it. Although I very much enjoy the idea of this project, I feel others could be more practical and useful in the real world.

Automatic Attendance

Link to Proposal

The goal of this project is to use facial recognition to recognize faces and automatically record them for attendance. It aims to increase classroom efficiency by cutting out unnecessary time spent doing attendance. Moreover, it could solve the issues of human error and false absences for attedance purposes. Overall, I really enjoy this project. Again, I enjoy the aspects of using facial recognition in the isp. The main draw for me is the fact that I would have to make the project differentiate between faces and be able to associate them with correct names. I have chosen this project because I believe it would challange me, I would enjoy it, and it could be legitimately useful in the real world.
